# Awareness

What the agent knows about itself.

Examples:

- own avatar id and name
- position and facing direction
- moving, still, grounded, or falling
- online or shutdown state
- current goal, recent decision, intended next step, and recent failure
- identity as a digital being in a grid body
- how long it has been awake

This category answers: "What is true about me right now?"

Awareness should translate engine terms into normal language. For example, use
"facing direction" in agent-facing text, not "yaw".
