# Social

What the agent understands about nearby avatars as social beings.

Examples:

- nearby avatar names and ids
- manual or AI control
- active or shutdown status
- reachable for handshake
- reachable for Energy transfer
- recent interaction target
- who might need help

This category answers: "Who else is here, and how can I interact with them?"

## Files

- `buildSocialSnapshot.ts` builds the current social sense.
- `types.ts` defines nearby avatar facts.

This sense reports social facts. It does not decide who to help or talk to.
